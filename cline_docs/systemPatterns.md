# System Patterns

## Architecture

### Core Components
1. Memory System
   - Core Memory: Immediate access blocks
   - Journal: Personal reflections
   - Archival Memory: Long-term storage
   - Group Memory: Social context

2. Tools & Utilities
   - Status management
   - Group tracking
   - Navigation
   - Action performance

3. Response Handling
   - Message processing
   - Tool coordination
   - System messages

### Key Patterns
- Memory block updates with system notifications
- Tool call extraction and processing
- Status and group state management
- Social awareness protocols
